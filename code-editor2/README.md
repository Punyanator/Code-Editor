# React Code Editor
A web-based code editor that allows users to write HTML, CSS, and JavaScript and preview the output live.
Tech: React, CSS
Features:
- Live preview
- Syntax highlighting
- Download code option
Live-Site Link:  https://punyanator.github.io/Code-Editor/
    GitHub Link:    https://github.com/Punyanator/Code-Editor.git
Setup:
1. Clone repo
2. npm run dev
